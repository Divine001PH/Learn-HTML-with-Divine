# Basics/Introduction
What is HTML?

HTML (HyperText Markup Language) is the standard language used to create the structure of web pages.

HTML tells the browser:

What content to display
How the content is organized

Think of HTML as the skeleton of a website.


| Tag               | Purpose                                |
| ----------------- | -------------------------------------- |
| `<!DOCTYPE html>` | Declares HTML5 document                |
| `<html>`          | Root element of the webpage            |
| `<head>`          | Contains metadata and page information |
| `<title>`         | Text displayed on the browser tab      |
| `<body>`          | Contains visible webpage content       |


Links : Used to navigate to another page or website.
<a href="https://example.com">Visit Website</a>


Images
Used to display pictures.
<img src="image.jpg" alt="Description">
src = image location
alt = alternative text if the image cannot load


Lists
Unordered List
<ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
</ul>
Ordered List
<ol>
    <li>Learn HTML</li>
    <li>Learn CSS</li>
    <li>Learn JavaScript</li>
</ol>

Line Break
Moves text to a new line.
<br>

Horizontal Line
Creates a horizontal divider.
<hr>

Text Formatting
<strong>Bold text</strong>
<em>Italic text</em>
<mark>Highlighted text</mark>


Comments
Comments are ignored by the browser.
<!-- This is a comment -->
Used for notes and organizing code.

Forms
Forms collect user input.
<form>
    <input type="text" placeholder="Enter your name">
    <button>Submit</button>
</form>

Common form elements:
input
textarea
select
button
HTML Attributes

Attributes provide extra information to elements.

Example:

<a href="https://google.com">Google</a>

Here, href is an attribute.

Another example:

<img src="photo.jpg" alt="My Photo">

src and alt are attributes.

Key Rule
Most HTML elements have:
<tagname>Content</tagname>

Example:

<p>Hello</p>
Opening tag: <p>
Content: Hello
Closing tag: </p>
First Mini Project Structure

A simple webpage should contain:

<body>
    <h1>My Portfolio</h1>

    <p>Welcome to my website.</p>

    <h2>About Me</h2>

    <p>I am learning web development.</p>

    <h2>Contact</h2>

    <a href="mailto:example@email.com">Email Me</a>
</body>

These fundamentals—document structure, headings, paragraphs, links, images, lists, forms, attributes, and comments—form the foundation of HTML. Once you're comfortable with them, the next step is learning semantic HTML (header, nav, section, article, footer) before moving on to CSS.
